# Gallifreyan
A graphics editor for visualizing sentences in Doctor's Cot Gallifreyan, a writing system created by Brittany Goodman and inspired by the fictional Gallifreyan language from *Doctor Who*.

🔗 [Official Website (now archived)](https://doctorscotgallifreyan.com/)

## Features

### 🔠 Multiple-Token Support
Enter and visualize entire sentences, including words, numbers, and punctuation marks, in Doctor's Cot Gallifreyan.

🔗 [Guide to the Writing System](https://doctorscotgallifreyan.com/learn-the-system) ([Backup Link](https://web.archive.org/web/20240220184555/http://doctorscotgallifreyan.com/learn-the-system))

<img alt="^256 Butterflies! Example" src="https://www.dropbox.com/scl/fi/bk5csxnqq6br0lt9nav95/256_butterflies.png?rlkey=kmzjx9dfccedqmb150nfnlsdg&st=81i54aia&raw=1" width="500">

### ✏️ Resizable Elements
- Resize words by dragging the outer border of the largest syllable.
- Adjust the word’s outer circle by dragging its border.
- Resize a syllable - along with all consecutive syllables -  by dragging its outer border.
- Resize a number or a mark by dragging its outer border.
- Resize digits by dragging their inner borders.
- Resize inner circles of syllables or marks by dragging their borders.

<img alt="Word Resizing" src="https://www.dropbox.com/scl/fi/jvqk5vktuc8ibfspjn9r8/word_resizing.gif?rlkey=tntx6v1sstgwuvkmy16l197pz&st=jumhnkqh&raw=1" width="500"> <img alt="Number Resizing" src="https://www.dropbox.com/scl/fi/jz74gef6cb677cmk2lu4y/number_resizing.gif?rlkey=22mdfh8oiy5dynbf3831gzuvr&st=acnomg0x&raw=1" width="500">

### 🎯 Element Positioning
Drag various components, like words, syllables, vowels, consonants, numbers, digits, and number marks, to reposition them.

<img alt="Word Repositioning" src="https://www.dropbox.com/scl/fi/hvqxfhoqyoz0kuk0prdwz/word_repositioning.gif?rlkey=f0c1gizjuxzhpy1rfif5ulkhf&st=d58sfajd&raw=1" width="500"> <img alt="Number Repositioning" src="https://www.dropbox.com/scl/fi/peh6tfvmozdglbe30nnh9/number_repositioning.gif?rlkey=on4635pogzkctswintmarerl2&st=j6i2n811&raw=1" width="500">

### 🎞️ Animation Support
- Toggle animation via the **Animation** button.
- Customize frame count and timing.

<img alt="Animation Example" src="https://www.dropbox.com/scl/fi/rqrytn8wto3ejkeqpd80c/animation.gif?rlkey=8je7xruel2lk8arg0agrn4g69&st=kop7nrm2&raw=1" width="500">

### 📤 Image Export
- Export designs as **PNG** or **animated GIF** files.
- Go to **File → Export as...**

<img alt="Export Example" src="https://www.dropbox.com/scl/fi/uzf092thlohtoo6sfx4k8/export.png?rlkey=3d77xnz5piqrnw3711dosmcza&st=tvaom9ul&raw=1" width="500">

### 🎨 Color Scheme Customization
- Choose custom colors for various components.
- Access via **Tools → Color Scheme Selector**.
- Click **Apply** to update or **Reset** to restore defaults.

<img alt="Color Scheme Selector" src="https://www.dropbox.com/scl/fi/cstj7cn8f5falhcuyh206/color_scheme_selector.png?rlkey=07sd6abnp14drlfax6edum8pp&st=hut6nwkm&raw=1" width="500">

## 🛠️ How to Run
Ensure **Python 3.10+** is installed.

1️⃣ Install dependencies:
```sh
pip install -r requirements.txt
```
2️⃣ Run the program:
```sh
python -m src.main
```
